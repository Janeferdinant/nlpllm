NumPy: Numerical Python

NumPy is the fundamental package for scientific computing with Python, providing support for large, multi-dimensional arrays and matrices, along with a large collection of high-level mathematical functions to operate on these arrays.

Import Convention: import numpy as np

Key Features

  • N-dimensional array object (ndarray)
  • Broadcasting functions for vectorized operations
  • Tools for integrating C/C++ and Fortran code
  • Linear algebra, Fourier transform, and random number capabilities
  • High-performance mathematical operations

Example Code

import numpy as np # Create arrays arr1 = np.array([1, 2, 3, 4, 5]) arr2 = np.zeros((3, 3)) arr3 = np.random.rand(2, 4) # Array operations print(f"Mean: {arr1.mean()}") print(f"Sum: {arr1.sum()}") print(f"Standard deviation: {arr1.std()}") # Mathematical operations print(f"Element-wise multiplication: {arr1 * 2}") print(f"Square root: {np.sqrt(arr1)}") # Advanced operations matrix1 = np.array([[1, 2], [3, 4]]) matrix2 = np.array([[5, 6], [7, 8]]) print(f"Matrix multiplication: \n{np.matmul(matrix1, matrix2)}")
Pro Tip: NumPy operations are much faster than equivalent Python loops because they're implemented in C.

When to Use NumPy

  • Performing mathematical operations on arrays or matrices
  • Working with large datasets in array form
  • Implementing numerical algorithms
  • As a foundation for other scientific libraries

Pandas: Python Data Analysis

Pandas is a powerful data manipulation and analysis library, offering data structures and operations for manipulating numerical tables and time series.

Import Convention: import pandas as pd

Key Features

  • DataFrame object for tabular data manipulation
  • Series object for one-dimensional labeled arrays
  • Tools for reading/writing data in different formats (CSV, Excel, SQL DBs)
  • Data alignment and handling of missing data
  • Powerful grouping and aggregation functionality
  • Time-series functionality

Example Code

import pandas as pd import numpy as np # Create a DataFrame data = { 'Name': ['Alice', 'Bob', 'Charlie', 'David', 'Emma'], 'Age': [25, 30, 35, 40, 45], 'City': ['New York', 'Boston', 'Chicago', 'Denver', 'San Francisco'], 'Salary': [50000, 60000, 70000, 80000, 90000] } df = pd.DataFrame(data) print(df.head()) # Basic operations print(f"\nDataFrame info:") print(df.info()) print(f"\nStatistical summary:") print(df.describe()) # Selection and filtering print(f"\nSelect people over 30:") print(df[df['Age'] > 30]) # Group by operation print(f"\nAverage salary by age group:") df['Age_Group'] = pd.cut(df['Age'], bins=[20, 30, 40, 50]) print(df.groupby('Age_Group')['Salary'].mean())
Pro Tip: Use the .loc accessor for label-based indexing and .iloc for integer-based indexing.

When to Use Pandas

  • Working with structured or tabular data
  • Cleaning and preprocessing data
  • Performing data analysis with statistical methods
  • Time series analysis
  • Working with labeled or indexed data

SciPy: Scientific Python

SciPy builds on NumPy and provides additional functionality for scientific and technical computing, with modules for optimization, linear algebra, integration, interpolation, and more.

Import Convention: from scipy import [submodule]

Key Features

  • Optimization and root finding (scipy.optimize)
  • Linear algebra operations (scipy.linalg)
  • Statistical functions (scipy.stats)
  • Numerical integration (scipy.integrate)
  • Signal processing (scipy.signal)
  • Image processing (scipy.ndimage)
  • Sparse matrix operations (scipy.sparse)

Example Code

import numpy as np from scipy import optimize, stats, integrate # Optimization example - finding minimum of a function def f(x): return x**2 + 10*np.sin(x) result = optimize.minimize(f, x0=0) print(f"Function minimum: {result.x[0]:.4f}") # Statistical example - t-test group1 = np.random.normal(5, 2, 100) group2 = np.random.normal(6, 2, 100) t_stat, p_value = stats.ttest_ind(group1, group2) print(f"T-test: t={t_stat:.4f}, p={p_value:.4f}") # Integration example def g(x): return np.sin(x) integral, error = integrate.quad(g, 0, np.pi) print(f"Integral of sin(x) from 0 to Ï€: {integral:.4f} (error: {error:.8f})")
Pro Tip: SciPy is organized into subpackages that cover different scientific computing domains. Import only what you need.

When to Use SciPy

  • Advanced scientific computations beyond basic NumPy
  • Statistical analysis and hypothesis testing
  • Solving optimization problems
  • Signal or image processing tasks
  • Working with sparse matrices for large datasets

Matplotlib: Visualization Library

Matplotlib is a comprehensive library for creating static, animated, and interactive visualizations in Python, providing a MATLAB-like interface.

Import Convention: import matplotlib.pyplot as plt

Key Features

  • Create publication-quality plots
  • Wide variety of plot types (line, scatter, bar, histogram, etc.)
  • Extensive customization options
  • Object-oriented API for embedded plots
  • Support for LaTeX formatting
  • Integration with Pandas and NumPy

Example Code

import matplotlib.pyplot as plt import numpy as np # Generate data x = np.linspace(0, 10, 100) y1 = np.sin(x) y2 = np.cos(x) # Create a figure with two subplots fig, (ax1, ax2) = plt.subplots(2, 1, figsize=(10, 8)) # First plot - Sine wave ax1.plot(x, y1, 'b-', linewidth=2, label='Sine') ax1.set_title('Sine Wave') ax1.set_xlabel('x') ax1.set_ylabel('sin(x)') ax1.grid(True) ax1.legend() # Second plot - Cosine wave ax2.plot(x, y2, 'r--', linewidth=2, label='Cosine') ax2.set_title('Cosine Wave') ax2.set_xlabel('x') ax2.set_ylabel('cos(x)') ax2.grid(True) ax2.legend() # Adjust layout plt.tight_layout()
Pro Tip: Use plt.subplots() for better control over multiple plots in a single figure.

When to Use Matplotlib

  • Creating static visualizations for reports or publications
  • Making custom plots not available in higher-level libraries
  • When you need fine-grained control over plot elements
  • When integrating with GUI applications
  • For creating scientific visualizations with precise specifications

Scikit-learn: Machine Learning in Python

Scikit-learn is a simple and efficient tool for data mining and data analysis, built on NumPy, SciPy, and matplotlib, focusing on machine learning applications.

Import Convention: from sklearn import [submodule]

Key Features

  • Supervised learning algorithms (classification, regression)
  • Unsupervised learning algorithms (clustering, dimensionality reduction)
  • Cross-validation and evaluation metrics
  • Feature extraction and preprocessing tools
  • Ensemble methods
  • Model selection and parameter tuning

Example Code

from sklearn.model_selection import train_test_split from sklearn.preprocessing import StandardScaler from sklearn.ensemble import RandomForestClassifier from sklearn.metrics import classification_report, confusion_matrix import numpy as np # Generate sample data X = np.random.rand(100, 4) # 100 samples, 4 features y = np.random.randint(0, 2, 100) # Binary classification labels # Split data into training and testing sets X_train, X_test, y_train, y_test = train_test_split(X, y, test_size=0.3, random_state=42) # Standardize features scaler = StandardScaler() X_train_scaled = scaler.fit_transform(X_train) X_test_scaled = scaler.transform(X_test) # Create and train the model model = RandomForestClassifier(n_estimators=100, random_state=42) model.fit(X_train_scaled, y_train) # Make predictions y_pred = model.predict(X_test_scaled) # Evaluate the model print("Confusion Matrix:") print(confusion_matrix(y_test, y_pred)) print("\nClassification Report:") print(classification_report(y_test, y_pred)) # Feature importance feature_importances = model.feature_importances_ print("\nFeature Importances:") for i, importance in enumerate(feature_importances): print(f"Feature {i+1}: {importance:.4f}")
Pro Tip: Use scikit-learn's pipeline feature to combine preprocessing steps and models into a single workflow.

When to Use Scikit-learn

  • Implementing machine learning models
  • Preprocessing and cleaning data for ML tasks
  • Evaluating model performance
  • Feature selection and engineering
  • Prototyping ML solutions before scaling up
