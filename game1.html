How to Use

  • Click and drag on the grid to create walls
  • Drag the green cell to set the start position
  • Drag the red cell to set the target position
  • Select an algorithm from the dropdown menu
  • Click "Start Visualization" to see the algorithm in action
  • Use "Generate Maze" to create maze patterns automatically
  • The dashboard will display metrics to help you understand each algorithm's performance

About the Algorithms

  • Dijkstra's Algorithm: Guarantees the shortest path but explores in all directions.
  • A* Search: Uses heuristics to guide the search toward the target, often more efficient than Dijkstra's.
  • Breadth-First Search: Explores nodes in layers, guarantees shortest path but may be less efficient.
  • Depth-First Search: Explores as far as possible along each branch before backtracking, doesn't guarantee shortest path.
